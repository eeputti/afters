# afters

Vite + React + Tailwind landing page for the **afters** brand.

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run preview
```

## Deploy to Vercel

This repository is ready for Vercel as a Vite project:

- **Framework preset:** `Vite`
- **Build command:** `npm run build`
- **Output directory:** `dist`
